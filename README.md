# Virelle Product Showcase

Virelle is a responsive product showcase website for a fashion and lifestyle brand. It presents clothing, accessories, fragrances, and skincare products with a clean and modern user interface.

## 🌟 Features
- Responsive design (mobile, tablet, desktop)
- Clean and modern product layout
- Interactive elements using JavaScript
- Smooth user experience
- Lightweight and fast loading..

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
